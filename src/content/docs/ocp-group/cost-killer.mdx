# Cost Killer (Opex vs Capex)

Dans l'industrie lourde (Mines, Chimie), les coûts sont massifs.

Aidez les équipes à structurer les coûts. Analysez la différence entre réduire les dépenses d'exploitation (Opex) et optimiser les investissements (Capex).

Si vous pouvez démontrer qu'une meilleure gestion de la sécurité (spécialité de DuPont) réduit les primes d'assurance et les coûts d'arrêts de travail, vous prouvez la valeur du consulting.
